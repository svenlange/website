Upgrade subversion on Ubuntu 12.04

This memo describes how to upgrade subversion on Ubuntu 12.04.

By default Ubuntu 12.04 comes with subversion 1.6. If you want to upgrade to 1.7 do the follwing on the console:

sudo apt-add-repository ppa:svn/ppa
sudo apt-get install subversion
svn --version --quiet

Done.
