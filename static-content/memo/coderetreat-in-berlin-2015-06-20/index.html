Coderetreat in Berlin 2015-06-20

This memo is about my experiences during the Coderetreat on the 20th June 2015 in Berlin.

The constraints for the iterations were pretty simple. Most of the attendees were debutants. This is what I have worked on in the iterations:

  1. TDD in plain Java
  2. Solving CGOL with Haskell and Pattern Matching
  3. BDD with Cucumber
  4. TDD in plain Java
  5. Implementation in Java and Unit Tests in Groovy (TDD)
  6. Solution Seeker

This was my second coderetreat in Berlin and it was fun again. Thanks to all the organizers and sponsors of the event.
