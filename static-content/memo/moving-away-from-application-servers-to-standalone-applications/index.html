Moving away from application servers to standalone applications

This memo shows why it makes sense to migrate away from big application servers with many embedded applications to bunch of standalone applications.

Still today many big companies have application servers with several applications deployed within them. These application servers come with an aweful amount of features and capabilities. On top their vendors like IBM or Oracle offer support contracts and give compatibility quaranties to other 3rd party systems like databases or messaging systems.

But there are also downsides to big application servers.
