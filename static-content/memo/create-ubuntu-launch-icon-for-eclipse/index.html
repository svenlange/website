Create Ubuntu launch icon for Eclipse

This memo describes how to create a launch icon for Eclipse under Ubuntu 14.04.

Open terminal and type:

sudo vi /usr/share/applications/eclipse.desktop

Insert and adjust the follwing content:

[Desktop Entry]
Name=Eclipse 4
Type=Application
Exec=/opt/eclipse/eclipse
Terminal=false
Icon=/opt/eclipse/icon.xpm
Comment=Integrated Development Environment
NoDisplay=false
Categories=Development;IDE;
Name[en]=Eclipse

Done.
