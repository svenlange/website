How to recursively remove Subversions .svn folders

To remove all the sometimes pretty annoying .svn folders here are some very helpful lines of code.

for /f "tokens=* delims=" %%i in ('dir /s /b /a:d *svn') do (
    rd /s /q "%%i"
)

Just place the above lines into a file called removeSvn.cmd, place it into the polluted folder and run it.
